Bienvenido a la documentación del Simulador ARMyC
=================================================

Contenido:

Este documento describe el simulador armyc, y como utilizarlo para el diseño de dispositivos a la medida.

Se detalla como construir un ambiente de desarrollo con los componentes necesarios para crear programas para la arquitectura ARM.

Debido a que ARMyC está pensado para ambientes UNIX y ha sido probado únicamente en Linux, el documento sigue la misma ideología.
